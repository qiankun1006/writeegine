## 📊 系统构成

```
骨骼动画系统
│
├─ 核心数学
│  └─ Matrix2D.js        (2D矩阵计算)
│
├─ 数据模型
│  ├─ Bone.js            (骨骼)
│  ├─ Skeleton.js        (骨骼系统)
│  ├─ Keyframe.js        (关键帧)
│  ├─ Track.js           (轨道)
│  └─ Animation.js       (动画)
│
├─ 播放控制
│  ├─ AnimationPlayer.js (播放器)
│  └─ SkeletonCommands.js (撤销重做)
│
├─ 用户界面
│  ├─ SkeletonAnimationEditor.js (主窗口)
│  ├─ Timeline.js                (时间轴)
│  ├─ BoneHierarchyPanel.js      (骨骼树)
│  └─ BonePropertyPanel.js       (属性编辑)
│
└─ 交互工具
   └─ BoneEditTool.js   (骨骼编辑工具)
```

## 🎓 工作流程

### 创建简单动画（5步）

```
1️⃣ 启动编辑器
   打开 /create-game-image 页面

2️⃣ 创建骨骼
   点击 🦴 按钮
   在画布上点击创建根骨骼
   继续点击添加子骨骼

3️⃣ 打开动画编辑器
   点击 🎬 按钮
   看到编辑面板出现

4️⃣ 制作关键帧
   在时间轴选择帧位置
   修改骨骼属性
   系统自动创建关键帧

5️⃣ 预览动画
   点击播放按钮
   观看骨骼动画效果
```
