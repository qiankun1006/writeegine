📚 使用说明

组件特性

  • 6种游戏场景效果：极简纯色、游戏血条、能量条、加载进度、复古像素、玻璃拟态
  • 完整配置选项：支持颜色、尺寸、边框、动画速度等全方位配置
  • 平滑动画：支持0.2s-1s可调的平滑过渡动画
  • 交互功能：支持拖拽调整、悬停提示、状态切换
  • 响应式设计：完美适配PC、平板和移动端
