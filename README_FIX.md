## 🏗️ 技术改动

### 修改的文件
```
src/main/resources/static/js/image-editor/MenuManager.js
```

### 改动内容
- 重写了图片导入逻辑（第 147-242 行）
- 从直接绘制改为通过图层系统
- 添加详细的日志和错误处理
- 代码从 60 行增加到 210+ 行

### 关键改进
```javascript
// 旧的错误方式 ❌
ctx.drawImage(img, 0, 0);
editor.render();  // 会覆盖图片

// 新的正确方式 ✅
const layer = new Layer({width, height});
layer.getContext().drawImage(img, 0, 0);
editor.document.addLayer(layer);
editor.render();  // 正确渲染图层
```
