# 骨骼动画功能 - 快速开始指南

## 🎬 功能入口

### 方式1：工具栏按钮（推荐）
1. 打开图片编辑器页面：`/create-game-image`
2. 在工具栏中找到骨骼动画工具按钮：
   - **🦴 骨骼编辑** - 用于创建和编辑骨骼
   - **🎬 骨骼动画编辑器** - 用于制作关键帧动画

### 方式2：菜单栏
1. 打开图片编辑器页面
2. 点击菜单栏的**动画**菜单
3. 会显示骨骼动画功能的说明和快捷键

## 🦴 骨骼编辑工具使用

### 激活工具
- 点击工具栏中的 **🦴** 按钮
- 快捷键：`Shift+B`

### 基本操作

#### 创建骨骼
1. 在画布中点击以创建根骨骼
2. 继续点击以添加子骨骼
3. 每个新的骨骼都会自动成为上一个骨骼的子骨骼

#### 选择骨骼
- 在画布中点击骨骼节点以选择它
- 选中的骨骼会以高亮显示

#### 移动骨骼
- 选中骨骼后，拖动节点以移动骨骼位置
- 支持相对于父骨骼的局部变换

#### 旋转骨骼
- 鼠标悬停在骨骼上查看旋转手柄
- 拖动旋转手柄来旋转骨骼
- 使用 Alt + 拖动 可以快速旋转

#### 修改属性
- 在右侧属性面板中查看和编辑骨骼属性
- 支持修改：位置(X, Y)、旋转(Rotation)、缩放(Scale)

## 🎬 骨骼动画编辑器使用

### 打开编辑器
- 点击工具栏中的 **🎬** 按钮
- 快捷键：`Shift+A`
- 编辑器将在页面底部或右侧弹出

### 动画编辑器界面

编辑器包含以下主要组件：

#### 1. 骨骼层级面板（左侧）
- 显示骨骼的树形结构
- 可以选择要编辑的骨骼
- 支持展开/折叠骨骼层级

#### 2. 时间轴编辑器（底部）
- 显示动画的时间刻度
- 可以调整动画总长度
- 支持改变帧速率（FPS）
- 可以拖动时间指针预览动画

#### 3. 关键帧轨道面板
- 显示每个骨骼的位置、旋转、缩放轨道
- 关键帧显示为菱形 ◇
- 支持点击选择、拖拽移动、右键删除

#### 4. 属性编辑面板（右侧）
- 显示选中骨骼在当前帧的属性
- 可以直接编辑位置、旋转、缩放值
- 支持选择插值类型：线性、贝塞尔曲线、步进

### 基本工作流

1. **准备骨骼**
   - 首先使用骨骼编辑工具创建角色的骨骼结构
   - 调整骨骼的位置和大小

2. **创建新动画**
   - 在编辑器中点击"新建动画"按钮
   - 设置动画的总帧数和帧速率

3. **创建关键帧**
   - 选择要编辑的骨骼
   - 在时间轴中的指定帧位置点击创建关键帧
   - 调整骨骼的位置、旋转、缩放
   - 关键帧会自动记录当前状态

4. **编辑动画**
   - 在时间轴中移动时间指针
   - 在不同的帧创建多个关键帧
   - 自动插值会生成中间帧

5. **预览和调整**
   - 点击"播放"按钮预览动画
   - 调整各帧的属性，实时看到效果
   - 可以调整播放速度

6. **导出动画**
   - 完成编辑后，可以导出为 JSON 文件
   - JSON 包含完整的骨骼数据和关键帧信息

## 📊 数据结构

### 骨骼数据
```javascript
{
  name: "骨骼名称",
  position: { x: 0, y: 0 },
  rotation: 0,          // 弧度
  scale: { x: 1, y: 1 },
  children: [],         // 子骨骼列表
  parentId: null        // 父骨骼ID
}
```

### 关键帧数据
```javascript
{
  time: 10,             // 帧号
  boneId: "bone1",
  properties: {
    position: { x: 100, y: 200 },
    rotation: 0.5,
    scale: { x: 1, y: 1 }
  },
  interpolation: "linear"  // linear | bezier | step
}
```

### 动画数据
```javascript
{
  name: "walk",
  duration: 60,         // 总帧数
  fps: 30,              // 帧速率
  tracks: {
    "bone1": { position: [...], rotation: [...], scale: [...] }
  }
}
```

## 🎮 快捷键

| 快捷键 | 功能 |
|--------|------|
| Shift+B | 激活骨骼编辑工具 |
| Shift+A | 打开/关闭动画编辑器 |
| Space | 播放/暂停动画 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Delete | 删除选中的骨骼或关键帧 |

## 🐛 常见问题

### Q1: 为什么看不到骨骼编辑按钮？
A: 确保页面已完全加载。检查浏览器控制台是否有错误信息。可能是某些脚本未正确加载。

### Q2: 如何确定我创建的骨骼有效？
A: 在编辑器中查看骨骼层级面板。有效的骨骼应该显示在树状结构中。

### Q3: 动画预览时为什么看不到效果？
A:
1. 确保已创建关键帧（时间轴中应该有菱形 ◇ 标记）
2. 确保帧速率设置正确
3. 检查骨骼是否有有效的位置/旋转/缩放数据

### Q4: 如何导出动画用于其他项目？
A: 使用编辑器的导出功能，可以生成包含骨骼和动画数据的 JSON 文件。
