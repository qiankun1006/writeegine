# 代码编辑器快速启动指南

## 🚀 快速开始

### 1. 访问编辑器
```
首页 → 创作游戏 → 代码开发
或直接访问：http://localhost:8083/create-game/code
```

### 2. 基本操作
- **编辑代码**: 在左侧编辑区输入 Java 代码
- **搜索符号**: 在右侧"代码索引"搜索框搜索类/方法
- **保存文件**: Ctrl+S 或点击"💾 保存"按钮
- **格式化**: 点击"✨ 格式化"按钮

## 📌 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存 |
| `Ctrl+/` | 注释 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 替换 |
| `Ctrl+G` | 跳转行 |

## 🎯 5 分钟学会

### 场景 1: 创建新 Java 类
```
1. 点击 "+ 新文件"
2. 输入文件名：MyClass
3. 编辑器自动生成类模板
4. Ctrl+S 保存
```

### 场景 2: 快速找到方法
```
1. 在代码索引中搜索 "main"
2. 点击搜索结果 "main"
3. 自动跳转到 main 方法
```

### 场景 3: 快速插入方法
```
1. 点击"新建方法"按钮
2. 方法模板插入到光标位置
3. 继续编辑方法体
```

## 💡 建议

### 最佳实践
- ✅ 经常保存 (Ctrl+S)
- ✅ 使用代码模板加速开发
- ✅ 利用代码索引快速导航
- ✅ 使用格式化保持代码整洁

### 性能建议
- ✅ 一个项目不超过 50 个文件
- ✅ 单文件代码不超过 5000 行
- ✅ 定期清理不需要的文件

## ❓ 常见操作

### Q: 如何删除文件？
A: 点击文件标签页的 ✕ 按钮（Main.java 无法删除）

### Q: 如何创建新文件？
A: 点击 "+ 新文件" 按钮，输入文件名

### Q: 代码会丢失吗？
A: 不会。代码保存在浏览器 LocalStorage，清除缓存才会丢失

### Q: 可以编译执行代码吗？
A: 目前版本不支持（未来版本计划支持）
