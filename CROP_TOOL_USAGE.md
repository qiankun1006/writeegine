# ✂️ 裁剪工具使用指南

## 功能概述

裁剪工具支持**两种方式**裁剪图像：

### 方式1️⃣: 矩形拖拽选择（类似矩形选择工具）

#### 步骤：
1. 点击工具栏的 **✂️ 裁剪** 按钮或按 **`C`** 快捷键
2. 在画布上**拖拽鼠标**创建裁剪框
   - 按下鼠标并拖动 → 创建矩形选区
   - 松开鼠标 → 裁剪框固定
3. 调整裁剪框（可选）
   - 拖拽裁剪框的**8个控制点**（4个角 + 4条边中点）
   - 移动整个框：在框内任意位置拖拽（**未来版本支持**）
4. 查看右侧面板的坐标值
5. 点击 **"✓ 应用裁剪"** 按钮完成裁剪

#### 交互细节：
- **矩形框** - 蓝色边框，显示裁剪区域
- **网格** - 九宫格辅助线（摄影构图）
- **遮罩** - 框外区域半透明，高亮裁剪区域
- **控制点** - 8个小方块，可拖拽调整大小

---

### 方式2️⃣: 坐标精确输入

#### 步骤：
1. 激活裁剪工具（同上）
2. 在右侧浮动面板中输入坐标值：
   - **位置 X**: 裁剪框左上角的 X 坐标
   - **位置 Y**: 裁剪框左上角的 Y 坐标
   - **宽度**: 裁剪框的宽度（像素）
   - **高度**: 裁剪框的高度（像素）
3. 输入时实时预览（画布上的裁剪框会动态更新）
4. 点击 **"✓ 应用裁剪"** 完成

#### 输入框特性：
- ✅ **实时预览** - 输入时画布动态更新
- ✅ **双向同步** - 鼠标操作也会自动更新输入框
- ✅ **数值验证** - 宽高必须 > 0 才能应用

## 🎯 操作流程示例

### 例1: 通过拖拽裁剪

```
1. 按 C 键激活工具
2. 在画布上从 (100, 100) 拖拽到 (300, 300)
   → 创建 200x200 的正方形裁剪框
3. 右侧面板自动显示：X=100, Y=100, 宽=200, 高=200
4. 点击"应用裁剪"
```

### 例2: 通过输入坐标裁剪

```
1. 按 C 键激活工具
2. 在右侧输入框输入：
   - X = 50
   - Y = 50
   - 宽 = 300
   - 高 = 200
3. 画布实时显示 300x200 的裁剪框
4. 点击"应用裁剪"
```

### 例3: 混合操作

```
1. 拖拽创建初步裁剪框
2. 发现位置不对，手动调整 X 和 Y 值
3. 拖拽右侧边框调整宽度
4. 输入框中宽度值自动更新
5. 点击"应用裁剪"
```

## 💡 使用技巧

### 1. 构图辅助
- 使用九宫格网格辅助构图
- 将主体放在网格交点上（黄金分割）

### 2. 精确裁剪
- 如果需要精确尺寸，直接在输入框输入
- 例：输入宽=1024, 高=768 进行标准屏幕比例裁剪

### 3. 快速调整
- 拖拽控制点快速调整
- 用鼠标比用键盘输入更快

### 4. 对称裁剪
- 先拖拽创建大概范围
- 然后通过输入框精确调整位置使其居中

---

## ⚠️ 注意事项

### 数值限制
- **X, Y**: 必须 ≥ 0（位置不能在图层外）
- **宽, 高**: 必须 > 0（需要有实际尺寸）

### 无损质量
- 使用 Canvas ImageData API 直接操作像素
- 不会因缩放而丢失质量
- 支持透明通道保留

### 图层更新
- 裁剪后图层尺寸会改变
- 图层内容会被替换为裁剪区域
- 可以通过撤销恢复原图

## 🔄 与矩形选择工具的区别

| 特性 | 矩形选择 | 裁剪工具 |
|------|---------|---------|
| 目的 | 选定区域用于其他操作 | 直接裁剪图层 |
| 是否修改图层 | 否，只是选择 | 是，永久改变 |
| 输入坐标 | 否 | 是 ✅ |
| 可撤销 | 无需撤销 | 是 ✅ |
| 控制点 | 否 | 是 ✅ |

## 🐛 故障排除

### 问题1: 看不到裁剪框
**解决**:
- 确认已在画布上拖拽（不是只点击）
- 检查宽和高是否 > 0
- 硬刷新浏览器 (Ctrl+Shift+R)

### 问题2: 输入框数值不生效
**解决**:
- 确认宽和高 > 0
- 按 Enter 或点击其他位置确认输入
- 检查浏览器控制台是否有错误

### 问题3: 无法拖拽控制点
**解决**:
- 确认已创建裁剪框
- 将鼠标放在控制点上（会变成调整光标）
- 尝试硬刷新重新加载
