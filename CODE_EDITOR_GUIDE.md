# Java 代码编辑器使用指南

## 📍 访问地址
- 主入口：`http://localhost:8083/create-game`
- 代码编辑器：`http://localhost:8083/create-game/code`

## ✨ 功能特性

### 1. 代码编辑
- **Monaco Editor** - 专业级代码编辑器
- **语法高亮** - Java 语言语法高亮
- **自动缩进** - 智能自动缩进
- **代码格式化** - 一键格式化代码
- **括号匹配** - 彩色括号对匹配

### 2. 文件管理
- **多文件编辑** - 支持同时打开多个 Java 文件
- **文件创建** - 快速创建新的 Java 文件
- **文件关闭** - 支持关闭非主文件
- **文件树浏览** - 左侧文件浏览面板
- **本地存储** - 自动保存到浏览器 LocalStorage

### 3. 代码索引
- **符号搜索** - 快速搜索类和方法
- **代码大纲** - 显示所有类、方法、字段
- **快速导航** - 点击索引项直接跳转
- **智能解析** - 自动提取代码符号

### 4. 快速操作
- **新建类** - 生成类模板
- **新建方法** - 生成方法模板
- **新建 main** - 生成 main 方法
- **注释块** - 生成 JavaDoc 注释

## 📝 使用流程

### 1. 创建新文件
```
点击 "+ 新文件" → 输入文件名 → 创建
```

### 2. 编写代码
```
在编辑器中输入 Java 代码，支持语法高亮和自动完成
```

### 3. 搜索符号
```
在"代码索引"面板搜索框输入 → 快速定位类或方法
```

### 4. 保存文件
```
Ctrl+S 或点击 "保存" 按钮 → 文件自动保存到 LocalStorage
```

### 5. 格式化代码
```
点击 "✨ 格式化" 按钮 → 代码自动格式化
```

## 🔨 编辑器界面组成

### 左侧编辑区
- **编辑器标签页** - 显示所有打开的文件
- **Monaco Editor** - 主编辑区域，显示代码

### 右侧侧栏
- **文件浏览** - 项目中所有 Java 文件列表
- **代码索引** - 当前文件的类、方法、字段列表
  - 包含搜索功能
  - 快速导航到指定位置
- **快速操作** - 快速插入代码模板

### 状态栏
- 显示当前行列位置
- 显示编辑的文件名
- 显示文件编码
- 显示语言类型

## 💾 文件存储

### 本地存储
所有文件保存在浏览器 LocalStorage 中：
- 自动保存编辑内容
- 刷新页面后内容不丢失
- 默认文件：`Main.java`

### 保存方式
1. **自动保存** - 每次编辑后自动保存到 LocalStorage
2. **手动保存** - 点击 "💾 保存" 按钮或按 Ctrl+S

## 📊 代码索引功能详解

### 自动识别的符号类型

#### 1. 类定义
```java
public class MyClass {
    // 在索引中显示为：MyClass [class]
}
```

#### 2. 方法定义
```java
public void myMethod() {
    // 在索引中显示为：myMethod [method]
}
```

#### 3. 字段定义
```java
private String name;  // 在索引中显示为：name [field]
```

### 搜索索引
- 在搜索框输入关键词
- 实时过滤匹配的符号
- 支持大小写不敏感搜索

### 快速跳转
- 点击索引项直接跳转到定义位置
- 编辑器自动滚动并定位光标

## 🚀 代码模板

### 新建类模板
```java

    // 构造方法
    public NewClass() {

    }

    // 成员方法
    public void method() {

    }

```

### 新建方法模板
```java
public void newMethod() {

```

### 新建 main 方法
```java

    System.out.println("Hello, World!");

### JavaDoc 注释块

/**

```

## ⚠️ 注意事项

1. **Main.java 不可删除** - 系统保留的默认文件
2. **文件名要求** - 必须是有效的 Java 标识符
3. **本地存储限制** - 浏览器 LocalStorage 通常限制为 5-10MB
4. **清空缓存会丢失文件** - 清空浏览器缓存会导致所有编辑内容丢失

## 🔍 代码索引工作原理

### 扫描过程
1. 读取编辑器中的 Java 代码
2. 逐行扫描，识别类、方法、字段定义
3. 提取符号名称和行号
4. 渲染到索引面板

### 正则表达式匹配
- **类**: `^public\s+class\s+(\w+)`
- **方法**: `public|private|protected|static\s+.*\s+(\w+)\s*\(`
- **字段**: `^\s+(private|public|protected)\s+\w+\s+(\w+)`

## 🐛 常见问题

### Q: 关闭浏览器后代码会丢失吗？
A: 不会。代码保存在 LocalStorage 中，除非手动清除浏览器缓存。

### Q: 可以导出代码吗？
A: 当前版本不支持导出，但可以复制粘贴代码内容。

### Q: 支持哪些语言？
A: 目前只支持 Java 语言。

### Q: 文件数量有限制吗？
A: 理论上没有限制，但受浏览器 LocalStorage 容量限制。
