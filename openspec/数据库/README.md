## 📋 最佳实践清单

本地开发前检查：

- [ ] MySQL 已安装并运行
- [ ] 数据库 `write_engine` 已创建
- [ ] 所有表已导入（至少 8 个表）
- [ ] 字符集正确设置为 `utf8mb4_unicode_ci`
- [ ] `application-dev.properties` 已正确配置
- [ ] Spring Boot 可以成功启动
- [ ] 数据库连接测试通过

代码开发时：

- [ ] 使用连接池管理数据库连接
- [ ] 为查询条件的字段创建索引
- [ ] 使用 `@Transactional` 管理事务
- [ ] 在 Repository 层进行 Model↔Entity 转换
- [ ] 使用参数化查询防止 SQL 注入
- [ ] 记录操作日志用于审计
