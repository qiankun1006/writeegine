## 🎨 组件设计

### 1. 按钮组件

#### 主按钮 (.btn-primary)
```
┌─────────────────────┐
│  导出为PNG          │  ← 青蓝色渐变
│ #00d4ff → #0099cc  │
└─────────────────────┘
     ↓ Hover
   ↑ 上浮 -2px
   ✨ 发光增强
```

**CSS特性**:
- 渐变色 (135deg 对角线渐变)
- 悬停动画 (translateY -2px, 0.3s)
- 发光阴影 (0 6px 20px cyan-glow)
- 点击反馈 (translate 复位)

#### 危险按钮 (.btn-danger)
```
┌─────────────────────┐
│  清空画布           │  ← 红色渐变
│ #ff3333 → #cc0000  │
└─────────────────────┘
    ⚠️ 警告颜色
```

**用途**: 清空/删除操作

#### 次要按钮 (.btn-secondary)
```
┌─────────────────────┐
│  撤销 | 重做        │  ← 灰色透明
│ 50%灰色 + 透明     │
└─────────────────────┘
     ↓ Hover
   → 变青蓝色
```

**用途**: 常规操作

### 2. 面板设计

#### 玻璃态面板
```
效果组合:
├── 背景: rgba(26, 31, 46, 0.8)
├── 模糊: backdrop-filter: blur(10px)
├── 边框: 1px solid rgba(0, 212, 255, 0.1)
└── 阴影: 0 8px 32px rgba(0, 0, 0, 0.3)

视觉效果: 半透明毛玻璃 + 深色背景透出
```

#### 面板分类

**图块选择面板** (.tile-selector-panel)
- 宽度: 150px (固定)
- 背景: 深色半透明
- 内容: 2列网格布局
- 滚动条: 自定义青蓝色

**编辑面板** (.editor-panel)
- 宽度: 自适应 (flex: 1)
- 布局: 3区域垂直分布
  1. 头部 (当前选中 + 工具栏)
  2. 中部 (Canvas 最大化)
  3. 底部 (信息 + 提示)

### 4. 分隔符设计

#### 可拖动分隔符 (.resizable-divider)

**静止状态**:
```
│
│ ← 8px 宽
│ 渐变: 透明 → 青蓝 → 透明
│
```

**悬停状态**:
```
│║ ← 更明亮的青蓝渐变
│║ 发光增强
│║ cursor: col-resize
```

**拖动状态** (.active):
```
│║║ ← 最强发光
│║║ 发光阴影最大
│║║ 视觉反馈强烈
```

**CSS实现**:
```css
background: linear-gradient(90deg,
    transparent 0%,
    rgba(0, 212, 255, 0.3) 50%,  /* 静止 */
    transparent 100%);

/* 悬停 */
background: linear-gradient(90deg,
    transparent 0%,
    rgba(0, 212, 255, 0.6) 50%,  /* 增强50% */
    transparent 100%);

/* 拖动 */
background: linear-gradient(90deg,
    transparent 0%,
    rgba(0, 212, 255, 0.8) 50%,  /* 最强 */
    transparent 100%);
```

## 🎬 动画设计

### 按钮悬停动画
```
默认状态: scale(1) translateY(0)
悬停状态: scale(1) translateY(-2px)
       + box-shadow 增强
       + 过渡时间: 0.3s
```

### 对话框出现动画
```
初始状态: opacity(0) translateY(20px)
显示状态: opacity(1) translateY(0)
持续时间: 0.3s
缓动函数: ease-out
```

### 分隔符悬停动画
```
静止→悬停:
  背景亮度: 30% → 60%
  发光强度: 0.1 → 0.3
  过渡时间: 0.3s
```

## 🎯 设计原则

### 1. **深色沉浸**
- 使用深色背景减少眼睛疲劳
- 青蓝色强调色提升专业感
- 符合游戏引擎编辑器风格

### 2. **玻璃态美学**
- `backdrop-filter` 毛玻璃效果
- 半透明背景显示层级关系
- 现代极简设计风格

### 3. **完整反馈**
- 悬停状态有视觉反馈
- 活跃状态有高亮显示
- 操作完成有确认提示

### 4. **响应式布局**
- 自适应不同屏幕尺寸
- Canvas 优先占用空间
- 侧边栏可拖动调整

### 5. **游戏化交互**
- 渐变色按钮 (像游戏UI)
- 发光阴影效果
- 符号化提示 (emoji 图标)
- 流畅动画过渡
