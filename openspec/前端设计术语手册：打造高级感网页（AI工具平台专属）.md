# 一、整体风格术语（AI工具平台专属版）

### 1. 科技极简主义（Tech Minimalism）

- 干净、留白多、元素少、信息清晰，突出AI工具核心功能，不冗余

- 适合：AI编码工具、AI教育工具、AI设计工具等各类AI平台前端

- 你可以说：**整体极简、留白多、干净高级，突出工具核心功能**

### 2. 玻璃拟态（Glassmorphism）

- 半透明、模糊背景、轻微边框、悬浮感，适配AI工具的科技感调性

- 适合：功能卡片、操作弹窗、侧边工具栏、顶部导航栏

- 你可以说：**功能卡片用玻璃拟态，半透明毛玻璃效果，凸显科技感**

### 3. 新拟态（Neumorphism）

- 柔和凸起/凹陷、同色系阴影、质感柔和，兼顾操作手感与高级感

- 适合：核心操作按钮、输入框、功能面板、参数调节模块

- 你可以说：**核心按钮做新拟态，轻微凸起，阴影柔和，操作有质感**

### 4. 轻奢现代风（Light Luxury）

- 低饱和、细线条、精致点缀，适配高端AI工具，不浮夸不简陋

- 你可以说：**轻奢风，低调高级，不花哨，贴合AI工具的专业调性**

### 5. 暗黑精致模式（Refined Dark Mode）

- 不是纯黑，是多层灰：深灰 → 灰 → 浅灰，适配长时间使用AI工具的场景

- 文字分层：白 → 浅白灰 → 淡灰，兼顾可读性与高级感

- 你可以说：**暗黑模式但不刺眼，层次细腻，适合长时间使用AI工具**

### 6. 扁平化 2.0（Flat 2.0）

- 扁平 + 轻微阴影 + 微渐变，现代、通用、耐看，适配各类AI工具

- 适配：基础功能模块、辅助操作区，不抢核心功能风头

- 你可以说：**偏扁平，但加一点细节质感，适配AI工具的简洁需求**

# 二、交互反馈术语（AI工具平台适配版）

### 1. 按钮状态（核心操作适配）

- **Hover 上浮**：鼠标放上去轻微上移、阴影变大，明确可点击提示

- **Hover 放大**：scale(1.05)，核心功能按钮专属，强化视觉引导

- **点击下沉**：点击时往下移 1–2px，模拟真实按压感，操作有确认感

- **点击波纹**：Material 那种水波纹，适配次要操作按钮，不突兀

- **禁用态**：透明度降低、灰色、不可点，明确功能不可用状态（如AI加载中）

- 你可以说：**核心按钮 hover 上浮放大，点击下沉，加波纹反馈；次要按钮简化反馈，不抢焦点**

### 2. 输入框交互（AI工具高频场景）

- **聚焦高亮**：聚焦时边框变色、轻微发光，明确当前输入区域（如AIprompt输入框）

- **错误抖动**：输入错误（如格式错误）轻微左右抖，友好提示修正

- **成功对勾动画**：提交成功（如提交prompt）出现对勾，强化操作反馈

- 你可以说：**输入框（尤其是AIprompt输入框）聚焦高亮，错误抖动，成功动画，提升操作体验**

### 3. 加载动画（AI工具核心场景）

- **骨架屏（Skeleton）**：AI生成内容未加载时，显示灰色占位条（如代码生成、文本生成区域）

- **淡入加载**：AI生成内容加载完平滑出现，避免生硬闪屏

- **旋转加载**：简约小圈，适配AI加载中（如生成中、处理中）状态，不花哨

- 你可以说：**AI生成区域用骨架屏加载，加载中显示简约旋转小圈，内容淡入，提升等待体验**

### 4. 卡片交互（功能模块适配）

- **卡片悬停上浮**：功能卡片（如AI工具模块、历史记录卡片）hover上浮，明确可交互

- **卡片阴影渐变**：悬停时阴影渐变加深，增强层次感

- **卡片淡入**：页面加载时卡片淡入，整体过渡自然

- 你可以说：**功能卡片 hover 上浮，阴影柔和渐变，页面加载时卡片淡入，整体流畅**

### 5. 通用微动效（全页面适配）

- 淡入淡出、滑入、弹性结束，适配AI工具的简洁调性，不浮夸

- 过渡自然：0.2s–0.3s，操作无延迟感，不影响使用效率

- 你可以说：**所有交互加平滑过渡（0.2-0.3s），不要生硬，适配AI工具的使用节奏**

# 三、视觉元素术语（AI工具平台适配版）

### 1. 色彩术语（贴合AI科技感）

- **低饱和色系**：不刺眼、高级，适配长时间使用，不产生视觉疲劳

- **莫兰迪色系**：灰调颜色，适配高端AI工具，低调不浮夸

- **单色体系**：一个主色（如科技蓝、浅青）+ 深浅变化，强化品牌感

- **黑白灰+点缀色**：高级感最强，适配各类AI工具，不挑场景

- **渐变色**：轻渐变、同色系渐变，用于核心按钮、重点模块，不大面积使用

- 你可以说：**颜色低饱和，黑白灰为主，点缀一个科技感主色（如浅蓝、浅青），适配AI工具调性**

### 2. 字体术语（兼顾可读性与科技感）

- **无衬线字体**：现代干净（Inter、Roboto、思源黑体），适配AI工具的科技感，可读性强

- **字体层级**：标题粗、正文细、辅助灰，明确AI工具的功能层级（如标题、操作提示、辅助说明）

- **行高 1.5–1.6**：阅读舒服，适配AI生成内容（如代码、文本）的展示场景

- 你可以说：**字体用无衬线字体，清晰有层级，标题醒目，正文（尤其是AI生成内容）行高舒适，易阅读**

### 3. 图标术语（AI工具功能适配）

- **线性图标**：简约高级，适配AI工具的简洁调性，用于功能图标、操作图标

- **面性图标**：饱满，用于核心功能图标（如生成、保存、导出），强化识别

- **双色图标**：质感强，用于重点操作图标，不大面积使用

- **统一风格**：全部细线/全部面型，不混杂，保持整体整洁

- 你可以说：**图标统一线性风格（核心功能用面性），精致简约，贴合AI工具的科技感**

# 四、布局 & 排版术语（AI工具平台后端最需要）

### 1. 布局术语（适配AI工具功能分区）

- **居中布局**：核心内容（如AI生成区域、输入区域）居中，不贴边，视觉舒适

- **栅格布局（Grid）**：功能卡片、工具模块整齐排列，适配多功能展示

- **弹性布局（Flex）**：自适应排列，适配不同屏幕，核心功能不挤压

- **响应式**：手机/电脑自动适配，AI工具核心功能（输入、生成）在任何设备都好用

- **移动优先**：先做手机再适配电脑，确保移动端使用流畅

- 你可以说：**栅格布局，功能模块整齐，响应式设计，核心操作区域居中，适配AI工具多场景使用**

### 2. 留白术语（高级感关键，适配AI工具）

- **呼吸感**：元素之间留足空隙，尤其是AI输入区与生成区，不拥挤

- **内边距充足**：卡片、输入框内部不挤，操作区域有足够空间，提升使用体验

- **8px 基准间距**：所有间距是 8 的倍数，保持整体统一，不杂乱

- 你可以说：**多留白，有呼吸感，不要挤，尤其是AI输入区和生成区，操作空间充足**

### 3. 圆角术语（适配AI工具柔和调性）

- **大圆角**：12px 现代柔和，用于卡片、弹窗，适配AI工具的柔和感

- **中小圆角**：6–8px 通用，用于按钮、输入框，不生硬

- 你可以说：**统一圆角（卡片12px，按钮6-8px），风格柔和，贴合AI工具调性**

### 4. 阴影术语（适配AI工具高级感）

- **轻阴影**：柔和、不脏，用于卡片、按钮，增强层次感不突兀

- **多层阴影**：用于核心功能模块，更精致，凸显重点

- 你可以说：**阴影轻一点、柔和一点，核心模块用多层阴影，凸显重点不浮夸**
