# 豆包模型接入指南

## 概述

本项目已集成了字节跳动火山引擎提供的豆包（Douban）大模型，用于AI立绘生成、图片生成等功能。豆包模型基于火山引擎的方舟（Ark）平台，提供了多个版本的模型供选择。

## 支持的豆包模型

### 1. **豆包专业版 5.0** (推荐用于生产环境)
- **模型ID**: `doubao-seedream-5-0-260128`
- **特点**: 最新版本，图片质量最高，支持最多参数选项
- **适用场景**: 高质量AI肖像生成、专业级图片生成
- **性能**: 生成速度快，质量优秀

### 2. **豆包轻量版 5.0**
- **模型ID**: `doubao-seedream-5-0-lite`
- **特点**: 轻量级模型，生成速度快，成本低
- **适用场景**: 开发测试、快速原型、成本敏感场景
- **性能**: 生成速度最快，成本最低

### 3. **豆包版本 4.0**
- **模型ID**: `doubao-seedream-4-0`
- **特点**: 稳定版本，功能完整
- **适用场景**: 兼容性要求高的场景

## 获取 API Key

### 步骤 1: 登录火山引擎控制台

访问 [https://console.volcengine.com/](https://console.volcengine.com/)，使用字节账户登录。

### 步骤 2: 进入方舟服务

在控制台中找到"方舟"（Ark）服务，进入方舟大模型服务平台。

### 步骤 3: 创建或查看 API Key

1. 进入 API 密钥管理页面
2. 点击"创建密钥"
3. 复制生成的 API Key
4. 将 API Key 填写到配置文件中

**参考链接**:
- [豆包模型官方文档](https://www.volcengine.com/docs/82379/1824121?redirect=1&lang=zh)
- [方舟 API 控制台](https://console.volcengine.com/ark/region:ark+cn-beijing/endpoint/detail?Id=ep-m-20260116205541-rnvnv&Tab=api&Type=preset)

## API 使用方式

### 1. 文生图 (Prompt to Image)

**最基础的用法**，通过文本提示词生成图片。

```java
// 在 AIPortraitController 中的使用示例
@PostMapping("/generate")
public ResponseEntity<GeneratePortraitResponse> generate(
    @RequestHeader("X-User-Id") Long userId,
    @Valid @RequestBody GeneratePortraitRequest request) {

    // request.getProvider() 为 "volcengine" 时使用豆包模型
    // request.getPrompt() 为生成的提示词
    // request.getGenerateCount() 为生成数量

    GeneratePortraitResponse response =
        aiPortraitService.createGenerationTask(userId, request);

    return ResponseEntity.status(HttpStatus.ACCEPTED).body(response);
}
```

```json

  "width": 1024,
  "height": 1024,
  "generateCount": 1,
  "seed": null

```

## 架构设计

### 服务架构

```
┌─────────────────────────────────────────┐
│   AIPortraitController (REST API)       │
├─────────────────────────────────────────┤
│   AIPortraitService (业务逻辑层)        │
├─────────────────────────────────────────┤
│   AIModelServiceFactory (工厂模式)      │
├─────────────────────────────────────────┤
│ ┌─────────────┐  ┌──────────────────┐  │
│ │VolcengineService  │ AliyunTongYiService  │
│ │  (豆包)      │  │  (通义)       │  │
│ └─────────────┘  └──────────────────┘  │
├─────────────────────────────────────────┤
│   ImageGenerationService (统一接口)     │
└─────────────────────────────────────────┘
```

### 核心类说明

| 类名 | 位置 | 功能 |
|------|------|------|
| `VolcengineService` | `service/` | 火山引擎豆包模型实现 |
| `AIModelServiceFactory` | `service/` | 模型工厂，支持多模型切换 |
| `AIPortraitService` | `service/` | AI立绘生成业务逻辑 |
| `AIPortraitController` | `controller/` | REST API 接口 |
| `GeneratePortraitRequest` | `dto/` | 生成请求DTO |
| `GeneratePortraitResponse` | `dto/` | 生成响应DTO |

## 支持的参数

### 文生图参数

| 参数名 | 类型 | 必需 | 描述 | 示例 |
|-------|------|------|------|------|
| prompt | String | ✓ | 生成提示词 | "年轻女性，长发..." |
| width | Integer | ✗ | 图片宽度 | 512, 1024 |
| height | Integer | ✗ | 图片高度 | 512, 1024 |
| count | Integer | ✗ | 生成数量 | 1-4 |
| seed | Long | ✗ | 随机种子（确定结果） | 12345 |

| 尺寸 | 像素 | 说明 |
|------|------|------|
| 512×512 | 262,144 | 小尺寸，快速 |
| 768×768 | 589,824 | 中等尺寸 |
| 1024×1024 | 1,048,576 | 标准尺寸，推荐 |
| 2K | 2048×2048 | 高清，需要更长时间 |
| 4K | 4096×4096 | 超高清，生成时间最长 |

## 错误处理

### 常见错误

| 错误码 | 含义 | 解决方案 |
|-------|------|--------|
| 400 | 请求参数错误 | 检查 prompt、尺寸等参数 |
| 401 | API Key 无效 | 检查 API Key 是否正确配置 |
| 429 | 请求过于频繁 | 实施限流、等待后重试 |
| 500 | 服务内部错误 | 检查日志，联系火山引擎支持 |
| 503 | 服务不可用 | 等待服务恢复 |

### 重试机制

系统内置了指数退避重试机制：

```properties
# 重试配置
volcengine.retry.max-attempts=3          # 最多重试3次
volcengine.retry.initial-delay=1000      # 初始延迟1秒
volcengine.retry.max-delay=10000         # 最大延迟10秒
volcengine.retry.multiplier=2            # 每次延迟翻倍
```

## 示例代码

### 完整的生成流程

```java
// 1. 创建请求
GeneratePortraitRequest request = new GeneratePortraitRequest();
request.setPrompt("一个年轻女性，蓝色眼睛，长金发");
request.setProvider("volcengine");
request.setModel("doubao-seedream-5-0-lite");
request.setWidth(1024);
request.setHeight(1024);
request.setGenerateCount(1);

// 2. 调用生成接口
ResponseEntity<GeneratePortraitResponse> response =
    portraitController.generate(userId, request);

// 3. 获取任务 ID
GeneratePortraitResponse body = response.getBody();
String taskId = body.getTaskId();

// 4. 轮询查询进度
while (true) {
    GenerateProgressResponse progress =
        portraitController.getProgress(userId, taskId).getBody();

    if (progress.getStatus().equals("completed")) {
        // 获取生成结果
        List<String> imageUrls = progress.getResultUrls();
        break;
    } else if (progress.getStatus().equals("failed")) {
        // 处理错误
        break;
    }

    Thread.sleep(2000);  // 等待 2 秒后再查询
}
```

## 故障排查

### 问题 1: API Key 无效或未配置

**症状**: 收到 401 错误或"API密钥无效"提示

**解决方案**:
1. 检查 `volcengine.ark.api.key` 配置是否正确
2. 验证 API Key 是否过期
3. 确保环境变量或配置文件被正确加载

### 问题 2: 请求超时

**症状**: 生成时间超过 timeout 配置

**解决方案**:
1. 增加 `volcengine.generate.timeout` 值
2. 检查网络连接
3. 尝试使用轻量级模型（`doubao-seedream-5-0-lite`）

### 问题 3: 生成质量差

**症状**: 生成的图片质量不理想

**解决方案**:
1. 改进提示词描述
2. 切换到专业版模型（`doubao-seedream-5-0-260128`）
3. 增加 `generateCount` 获取多个版本选择

## 最佳实践
