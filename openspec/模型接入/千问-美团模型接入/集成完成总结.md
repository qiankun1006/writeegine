## 🏗️ 架构设计

```
┌─────────────────────────────────────────┐
│   AIPortraitController (REST API)       │
├─────────────────────────────────────────┤
│   AIPortraitService (业务逻辑层)        │
├─────────────────────────────────────────┤
│   AIModelServiceFactory (工厂模式)      │
├─────────────────────────────────────────┤
│ ┌─────────────┐  ┌──────────────────┐  │
│ │VolcengineService  │ AliyunTongYiService  │  │

│ ┌─────────────────────────────────────┐ │
│ │     MeituanQwenService (千问)       │ │
│ │                                     │ │

│ │  - Qwen-Image-Edit-Meituan (图生图)│ │

├─────────────────────────────────────────┤
│   ImageGenerationService (统一接口)     │
└─────────────────────────────────────────┘
```

### 核心类说明

| 类名 | 位置 | 功能 |
|------|------|------|
| `MeituanQwenService` | `service/` | 千问-美团模型实现 |
| `AIModelServiceFactory` | `service/` | 模型工厂，支持多模型切换 |
| `ImageGenerationService` | `service/` | 统一图像生成接口 |

## 🚀 使用示例

### 基础使用
```java
@Autowired
private MeituanQwenService qwenService;

// 文生图
List<String> imageUrls = qwenService.generateImage(
    "一个年轻女性角色，长棕发，穿着蓝色和服",
    1024, 1024, 1, null
);

// 图生图
List<String> imageUrls = qwenService.generateFromImage(
    "把墙的颜色改成紫色，背景加上闪光灯效果",
    "http://example.com/reference.jpg",
    0.7, 1024, 1024
);
```

### 工厂模式使用
```java
@Autowired
private AIModelServiceFactory factory;

ImageGenerationService service = factory.getService("Qwen-Image-Meituan");
List<String> imageUrls = service.generateImage(prompt, width, height, count, seed);
```

### REST API 使用
```bash
curl -X POST http://localhost:8080/api/ai/portrait/generate \
  -H "Content-Type: application/json" \
  -H "X-User-Id: 123" \
  -d '{
    "prompt": "一个年轻女性角色，长棕发，穿着蓝色和服",
    "provider": "meituan",
    "model": "Qwen-Image-Meituan",
    "width": 1024,
    "height": 1024,
    "generateCount": 1
  }'
```
