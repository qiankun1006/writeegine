# 千问-美团模型快速启动指南

## ⚡ 5 分钟快速上手

### 第 1 步：获取 API Key（2 分钟）

1. 访问 [https://aigc.sankuai.com/](https://aigc.sankuai.com/)
2. 登录你的美团 AI 平台账户
3. 进入"API 管理"页面
4. 创建或复制 API Key

### 第 2 步：配置 API Key（1 分钟）

#### 开发环境
编辑 `src/main/resources/application-dev.properties`:
```properties
meituan.qwen.api.key=your-api-key-here
meituan.qwen.default.model=Qwen-Image-Meituan
```

#### 生产环境
编辑 `src/main/resources/application-prod.properties`:
```properties
meituan.qwen.api.key=your-production-api-key
meituan.qwen.default.model=Qwen-Image-Meituan
```

### 第 3 步：启动应用（1 分钟）

```bash
# 开发环境启动
mvn spring-boot:run

# 生产环境启动
mvn spring-boot:run -Pprod
```

### 第 4 步：调用 API（1 分钟）

```bash
# 创建文生图任务
curl -X POST http://localhost:8080/api/ai/portrait/generate \
  -H "Content-Type: application/json" \
  -H "X-User-Id: 123" \
  -d '{
    "prompt": "一个年轻女性角色，长棕发，穿着蓝色和服",
    "provider": "meituan",
    "model": "Qwen-Image-Meituan",
    "width": 1024,
    "height": 1024,
    "generateCount": 1
  }'

# 响应示例
# {
#   "taskId": "gen-20260318-xxxxx",
#   "status": "PENDING",
#   "createdAt": 1710768000000
# }
```

完成！🎉

## 核心概念

### 支持的模型

| 模型 | ID | 功能 | 推荐 |
|------|----|------|------|
| **千问图像生成** | `Qwen-Image-Meituan` | 文生图 | 角色设计、场景生成 |
| **千问图像编辑** | `Qwen-Image-Edit-Meituan` | 图生图 | 图片修改、风格转换 |

### 常用参数

```json
{
  "prompt": "生成的描述（必需）",
  "provider": "meituan",
  "model": "Qwen-Image-Meituan",
  "width": 1024,
  "height": 1024,
  "generateCount": 1,
  "seed": null
}
```

### API 流程

```
1. 创建任务 (POST /generate)
   ↓
2. 获取 taskId
   ↓
3. 轮询查询进度 (GET /progress/{taskId})
   ↓
4. 获取结果 (resultUrls)
```

## 故障排除

### 常见问题

#### 1. API Key 配置错误
**症状**: `生成失败: API请求失败: 401`
**解决方案**: 检查 `application-dev.properties` 中的 `meituan.qwen.api.key` 配置

#### 2. 服务未配置
**症状**: `千问-美团服务未配置，请检查API密钥`
**解决方案**:
1. 确认 API Key 已正确配置
2. 检查配置文件是否被正确加载
3. 重启应用

#### 3. 请求超时
**症状**: `生成失败: timeout`
**解决方案**:
1. 检查网络连接
2. 增加超时时间配置
3. 实现重试机制

#### 4. 参数错误
**症状**: `生成失败: API请求失败: 400`
**解决方案**:
1. 检查提示词是否符合要求
2. 确认尺寸参数在支持范围内
3. 验证其他参数格式

## 性能优化

### 选择合适的模型
- **开发/测试**: 使用 `Qwen-Image-Meituan`（通用模型）
- **生产环境**: 根据需求选择合适的模型
- **批量生成**: 考虑异步处理和队列机制

### 参数优化
- 合理选择图片尺寸（不是所有场景都需要 1024x1024）
- 控制生成数量
- 复用相同的种子值以获得稳定结果

### 缓存策略
```java
// 对相同提示词的生成结果进行缓存
@Cacheable(value = "qwenImages", key = "#prompt")
public List<String> cachedGeneration(String prompt) {
    return meituanQwenService.generateImage(prompt, 1024, 1024, 1, null);
}
```
