## Requirements
### Requirement: 骨骼编辑工具
系统 SHALL 提供骨骼选择和变换工具。

#### Scenario: 用户在画布上选择和旋转一个骨骼
- **WHEN** 用户编辑骨骼
- **THEN** 用户必须能够点击选择骨骼
- **AND** 用户必须能够拖拽移动骨骼
- **AND** 用户必须能够旋转骨骼（通过旋转手柄或数值输入）
- **AND** 选中的骨骼必须有视觉高亮

#### Scenario: 用户在画布上选择和旋转一个骨骼
```html
<!-- 用户流程 -->
1. 在工具栏点击"骨骼编辑工具"
2. 在画布上点击要编辑的骨骼（显示高亮）
3. 拖拽骨骼末端进行旋转
4. 属性面板实时显示骨骼的旋转值
```

### Requirement: 骨骼层级面板
系统 SHALL 显示和管理骨骼树结构。

#### Scenario: 用户在层级面板中调整骨骼关系
- **WHEN** 用户管理骨骼层级
- **THEN** 必须显示骨骼树的完整层级
- **AND** 用户必须能够展开/折叠树节点
- **AND** 用户必须能够在树中选择骨骼
- **AND** 用户必须能够拖拽改变骨骼的父子关系

#### Scenario: 用户在层级面板中调整骨骼关系
```
骨骼层级：
  [+] Root
    [+] Arm
      [-] Hand
    [+] Leg
```

### Requirement: 时间轴编辑器
系统 SHALL 提供时间轴界面用于关键帧编辑。

#### Scenario: 用户在时间轴上定位到第 30 帧
- **WHEN** 用户使用时间轴
- **THEN** 必须显示时间刻度（秒或帧数）
- **AND** 必须显示时间指针（可拖拽）
- **AND** 必须支持缩放和滚动时间轴
- **AND** 必须显示帧速率和总帧数

#### Scenario: 用户在时间轴上定位到第 30 帧
```
时间轴：0s ----[---|----]---- 2.5s
             时间指针在第 30 帧（1.25s，@24fps）
```

### Requirement: 关键帧轨道
系统 SHALL 显示和编辑关键帧。

#### Scenario: 用户在时间轴上设置关键帧
- **WHEN** 用户编辑关键帧
- **THEN** 每个骨骼必须有一个轨道，显示其关键帧
- **AND** 关键帧必须显示为可视化标记
- **AND** 用户必须能够点击关键帧选择
- **AND** 用户必须能够拖拽关键帧移动时间位置
- **AND** 用户必须能够右键删除关键帧

#### Scenario: 用户在时间轴上设置关键帧
```
骨骼: Arm
轨道: ◇ -------- ◇ -------- ◇
      0帧        30帧       60帧
```

### Requirement: 属性编辑面板
系统 SHALL 编辑选中骨骼或关键帧的属性。

#### Scenario: 用户通过属性面板调整骨骼位置
- **WHEN** 用户编辑属性
- **THEN** 必须显示骨骼的位置（x, y）、旋转、缩放
- **AND** 必须显示关键帧的插值类型
- **AND** 用户必须能够通过数值输入修改属性
- **AND** 修改必须实时更新预览

#### Scenario: 用户通过属性面板调整骨骼位置
```
选中骨骼: Arm
位置: X: [100] Y: [50]
旋转: [45.5] °
缩放: X: [1.0] Y: [1.0]
```

### Requirement: 工具栏和控制按钮
系统 SHALL 提供动画编辑的核心控制按钮。

#### Scenario: 用户点击播放按钮预览动画
- **WHEN** 用户使用工具栏
- **THEN** 必须有"新建骨骼"按钮
- **AND** 必须有"播放/暂停"按钮
- **AND** 必须有"停止"按钮
- **AND** 必须有"循环播放"开关
- **AND** 必须有播放速度调整滑块

#### Scenario: 用户点击播放按钮预览动画
```
[新建骨骼 ▼] [◀◀] [▶] [⏸] [■] [◉循环] 速度: [1.0x ▼]
```

### Requirement: 图片编辑器主界面
系统 SHALL 集成骨骼动画编辑模块到现有 UI。

#### Scenario: 用户打开骨骼动画编辑页面
- **WHEN** 用户使用骨骼动画编辑功能
- **THEN** 右侧面板必须添加"骨骼层级"和"骨骼属性"标签页
- **AND** 底部必须添加时间轴编辑器（可折叠）
- **AND** 工具栏必须添加骨骼编辑工具按钮

#### Scenario: 用户打开骨骼动画编辑页面
```
┌─────────────────────────────────────────┐
│ 工具栏 [选择] [骨骼✨] [笔刷] ...        │
├─────┬──────────────────┬──────────────┤
│左侧 │                  │  骨骼层级   │
│面板 │   Canvas         │  [+] Root   │
│     │   画布显示       │    [-] Arm  │
│     │   骨骼和图片     │    [-] Hand │
├─────┴──────────────────┴──────────────┤
│ 时间轴编辑器 [0.0s] ▶ ◇---◇---◇       │
└─────────────────────────────────────────┘
```

### Requirement: 参数 Hover 提示
系统 SHALL 为所有参数提供详细的中文说明。

#### Scenario: 用户 hover 在"硬度"参数上
- **WHEN** 用户查看参数说明
- **THEN** 骨骼编辑工具的参数必须有 hover 提示
- **AND** 属性输入框必须有 hover 提示
- **AND** 关键帧插值类型选择必须有说明
- **AND** 所有提示必须清晰、简洁，包含单位和推荐值

#### Scenario: 用户 hover 在"硬度"参数上
```
参数: 硬度 [?]
Tooltip: "骨骼变换的平滑程度。0=自由旋转，1=锁定方向。推荐值 0.3"
