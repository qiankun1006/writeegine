# 龙骨平台（DragonBones）源码解析

## 概述

DragonBones 是一个开源的 2D 骨骼动画运行时库，支持多种游戏引擎和框架。该源码库包含了完整的 JavaScript/TypeScript 运行时实现，以及针对多个主流游戏引擎的适配层。

## 目录结构总览

```
不是本项目功能，只是用来参考和学习的开源代码，不要改动/
├── DragonBones/           # DragonBones 核心库
├── Egret/                 # Egret 引擎适配
├── Pixi/                  # PixiJS 引擎适配
├── Phaser/                # Phaser 引擎适配
├── Hilo/                  # Hilo 引擎适配
├── Cocos/                 # Cocos Creator 引擎适配
├── ThreeJS/               # Three.js 引擎适配
├── builds/                # 构建工具
├── docs/                  # 文档资料
└── package.json           # 项目配置
```

## 各文件夹职责详解

### 1. DragonBones/ - 核心库

**职责：** DragonBones 的核心运行时库，提供骨骼动画的基础功能和数据结构。

**子目录结构：**
- `src/dragonBones/core/` - 核心基础类
  - `BaseObject.ts` - 基础对象类
  - `DragonBones.ts` - DragonBones 主类
- `src/dragonBones/geom/` - 几何变换相关
  - `Point.ts` - 点坐标
  - `Matrix.ts` - 矩阵变换
  - `Rectangle.ts` - 矩形区域
  - `Transform.ts` - 变换类
  - `ColorTransform.ts` - 颜色变换
- `src/dragonBones/event/` - 事件系统
  - `EventObject.ts` - 事件对象
  - `IEventDispatcher.ts` - 事件分发接口
- `src/dragonBones/model/` - 数据模型
  - `ArmatureData.ts` - 骨架数据
  - `AnimationData.ts` - 动画数据
  - `BoneData.ts` - 骨骼数据
  - `SlotData.ts` - 插槽数据
  - `SkinData.ts` - 皮肤数据
  - `TextureAtlasData.ts` - 纹理集数据
- `src/dragonBones/parser/` - 数据解析器
  - `DataParser.ts` - 数据解析基类
  - `BinaryDataParser.ts` - 二进制数据解析
  - `ObjectDataParser.ts` - 对象数据解析
- `src/dragonBones/factory/` - 工厂模式
  - `BaseFactory.ts` - 基础工厂类
- `src/dragonBones/armature/` - 骨架系统
  - `Armature.ts` - 骨架类
  - `Bone.ts` - 骨骼类
  - `Slot.ts` - 插槽类
  - `Constraint.ts` - 约束类
- `src/dragbones/animation/` - 动画系统
  - `Animation.ts` - 动画控制器
  - `AnimationState.ts` - 动画状态
  - `WorldClock.ts` - 世界时钟

## 架构设计特点

### 1. 模块化设计
- 核心库与引擎适配层分离
- 每个引擎独立维护，版本化管理
- 统一的接口规范

### 2. 工厂模式
- 使用工厂模式创建引擎特定的对象
- 统一的创建接口，不同的实现
- 便于扩展新的引擎支持

### 3. 事件驱动
- 完整的事件系统
- 支持动画事件、骨架事件等
- 松耦合的组件通信

### 4. 数据驱动
- 基于 JSON 的数据格式
- 支持二进制和文本格式
- 版本兼容性处理

## 使用方式

### 1. 直接使用
```bash
npm install dragonbones-runtime
dbr pixijs@8.0.0  # 生成 PixiJS 8.x 的 DragonBones 文件
```

### 2. 集成到项目
```typescript
import { DragonBones, PixiFactory } from "dragonbones-pixi";

const factory = new PixiFactory();
factory.parseDragonBonesData(dragonBonesData);
factory.parseTextureAtlasData(textureAtlasData, texture);

const armatureDisplay = factory.buildArmatureDisplay("armatureName");
```

## 总结

DragonBones 源码展现了优秀的游戏引擎架构设计：
1. **跨平台支持** - 支持多种主流游戏引擎
2. **版本管理** - 良好的版本兼容性
3. **模块化** - 清晰的模块划分和职责分离
4. **扩展性** - 易于添加新的引擎支持
5. **工具链完善** - 完整的构建和部署工具

这套架构为开发者提供了强大的 2D 骨骼动画解决方案，同时保持了良好的可维护性和扩展性。
