# 龙骨功能特性表

| 功能模块 | 功能名称 | 功能描述 |
| :--- | :--- | :--- |
| **核心动画制作** | 骨骼操作 | 为图片绑定骨骼以制作动画，使角色动作制作更方便、更逼真，动画更流畅。 |
| | 时间轴 | 动画制作的核心，可宏观调节各元件动画的关键帧、播放速度、动画补间等细节。 |
| | 曲线编辑器 | 通过调整贝塞尔曲线来控制两帧之间的差值，实现栩栩如生的动画效果。 |
| | 洋葱皮工具 | 可同时看到前后若干帧的影图，方便更精准地调节动画细节。 |
| **AI与智能生成** | AI生成动画 | 龙骨已接入DeepSeek。在大模型加持下，支持用一句话生成动画，让动画制作更简单。 |
| | 自动生成网格 | 根据图片的形状，自动生成边框和填充三角形。 |
| **变形与网格** | 网格和自由变形 | 在图片矩形边界内自定义多边形，提高纹理集空间使用率。通过移动网格点变形图片，实现扭曲、拉伸、转面等伪3D效果。 |
| | 网格变形器 | 实现类似Live2D的动画效果。 |
| | 矢量形状 | 支持矢量形状及形状变形动画。 |
| **骨骼与动力学** | IK骨骼约束 | 通过反向动力学为角色摆姿势，建立约束使动作编辑操控更方便，生成的动作更自然。 |
| | 骨骼权重和蒙皮动画 | 将网格点绑定骨骼权重，骨骼运动带动网格变形，产生图片整体的弯曲、飘动等效果。 |
| | 骨架嵌套 | 项目中可创建多个骨架并进行任意的嵌套重用。 |
| | 路径约束 | 支持路径约束，让骨骼沿着预定轨道前进。 |
| | 物理 | 支持物理约束，如重力、风力、惯性。 |
| | 变换约束 | 支持变换约束，可使骨骼跟随其他骨骼的旋转、平移、缩放。 |
| **遮罩与效果** | 位图遮罩 | 支持利用位图的透明通道进行遮罩，呈现羽化效果。 |
| | 形状遮罩 | 支持形状遮罩。 |
| **导入与导出** | 导入 | 支持导入PSD文件、图片、纹理集、龙骨数据格式、Spine等第三方动画格式。 |
| | 导出 | 支持导出纹理集、序列帧、龙骨动画数据、Spine动画数据等。 |
