## 技术亮点

### 1. 可复用菜单架构
MenuManager 是一个通用的菜单管理基类，提供了：
- 统一的菜单打开/关闭逻辑
- 事件处理和事件委托
- 自动位置调整算法
- 易于扩展的子类机制

这使得添加其他菜单（编辑、图像、图层等）变得非常简单。

### 2. 浏览器端文件处理
利用现代浏览器 API：
- `File API` 用于读取用户选择的文件
- `Blob API` 用于创建和导出文件
- `FileReader` 用于异步读取文件内容
- `canvas.toBlob()` 用于转换画布为二进制数据

这避免了对后端的依赖，提升了前端性能和用户体验。

### 3. 智能位置调整
`_adjustPosition()` 方法检测菜单是否超出视口，并自动调整：
- 右边界检测：超出时从右侧对齐而不是左侧
- 下边界检测：超出时菜单向上展开而不是向下
- 支持不同的屏幕尺寸和分辨率

### 4. 错误处理
完整的错误处理和用户反馈：
- 文件加载失败时显示 alert
- 导出失败时显示详细错误信息
- Console 日志用于调试
- 支持多种文件格式

## 依赖关系

- 无外部依赖（纯 JavaScript）
- 兼容所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 不依赖 jQuery 或其他框架
