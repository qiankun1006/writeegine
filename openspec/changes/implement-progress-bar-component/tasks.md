- [x] 1.3 实现6种进度条效果
  - 极简纯色款：纯色背景和填充，支持自定义颜色和边框
  - 游戏血条款：线性渐变，血滴纹理，受损闪烁动画，低血量警示
  - 能量条款：径向渐变，流光动画，充能脉冲效果
  - 加载进度款：条纹动效，进度数字显示，状态图标
  - 复古像素款：8bit像素边框，阶梯式填充，像素字体
  - 玻璃拟态款：毛玻璃背景，半透明填充，模糊边框
