# 原始用户需求

在 http://localhost:8083/create-game/image 页面实现一个游戏物品菜单绘图功能。

## 需求详情

### 功能作用域
- 要在图片编辑器(/create-game/image 页面)中新增的一个绘图功能

### 绘图功能的含义
- 可以画出图片高亮的样子（物品菜单中悬停高亮效果）

### 三种布局的关系
- 支持列表式、图标网格式、侧边栏式 3 种布局
- 用户可以选择绘制不同的物品菜单样式

### 四种风格的具体要求
- 支持像素风、暗黑风、卡通风、科幻风 4 种风格
- 每种风格都需要包括：
  - UI 框架(背景、边框、圆角等)
  - 文本样式(字体、大小、颜色)
  - 物品图标(大小、排列、阴影)
  - 悬停效果(高亮、放大、动画)

### 与现有系统的集成
- 新建文件（不集成到现有的 ImageEditor.js 或 MenuManager.js）
