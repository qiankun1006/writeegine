# 提案总结：游戏创作平台门户重构

## 📋 提案概览

**变更 ID**: `redesign-homepage-with-game-menu`
**实施阶段**: 提案阶段（待审批）
**关键决策**: 将首页从通用 Web 应用示例改造为游戏创作平台专业门户

---

## 🎯 核心目标

### 用户需求
用户想要一个现代化的游戏创作平台首页，提供三个主要功能入口：
1. **创作游戏** - 进入 Tilemap 编辑器创作游戏地图
2. **我的游戏** - 管理自己创建的所有游戏
3. **游戏广场** - 发现和浏览社区分享的游戏

### 设计理念
- 将 Tilemap 编辑器从独立工具 → 集成为创作游戏的子模块
- 统一使用深色科技感主题，打造专业的游戏平台视觉
- 提供完整的游戏创作工作流：创建 → 编辑 → 保存 → 发布

## ⚠️ 关键设计决策

### 决策 1：路由结构
```
/                    首页门户
/create-game         创作游戏（新建或编辑）
/my-games            我的游戏列表
/game-plaza          游戏广场
/tilemap-editor      编辑器独立模式（向后兼容）
```

### 决策 2：编辑器集成方式
```
独立模式：  /tilemap-editor
  - 编辑器在独立页面中显示
  - 主要用于导出 PNG

集成模式：  /create-game
  - 编辑器在创作页面中嵌入
  - 与游戏信息结合
  - 主要用于保存游戏到数据库
```

### 决策 3：主题继承
```
所有新页面采用深色科技感主题：
  - 背景：#0f1419 → #1a1f2e（渐变）
  - 强调色：#00d4ff（青蓝）
  - 文字：#e8e9ea（亮灰白）
  - 效果：玻璃态 + 发光 + 动画
```
