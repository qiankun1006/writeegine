# 🚀 快速开始指南

## 提案概览（5 分钟）

### 是什么？
这是一份关于 **AI 人物立绘生成器前端布局重构** 的 OpenSpec 提案。目标是将复杂的多栏响应式布局简化为清晰的 **左右分栏布局**。

### 为什么？
- ✅ 简化用户界面，降低认知负担
- ✅ 改进 PC 端体验（1200px+）
- ✅ 提升代码可维护性
- ✅ 为后续功能扩展预留接口

### 改什么？
- 🎨 **布局**：从 3-4 栏 → 2 栏（左 380px 固定 + 右自适应）
- 📝 **左侧**：提示词 + 参考图片 + 参数配置（可折叠）
- 👁️ **右侧**：预览区（90%）+ 操作区（10%）
- 🎯 **样式**：浅灰/纯白 + 蓝色按钮 + 8px 圆角
- ⚙️ **技术**：Vue 3 + Element Plus + SCSS 主题变量

## 🎯 核心决策一览

| 决策 | 选择 | 理由 |
|------|------|------|
| 布局方案 | Flexbox 双栏 | 简洁、浏览器兼容好、代码直观 |
| UI 库 | Element Plus | 官方组件、文档全、团队熟悉 |
| 框架语法 | Vue 3 `<script setup>` | 现代化、代码简洁、性能优化 |
| 样式系统 | SCSS + 主题变量 | 易于维护、支持复用、主题统一 |
| 状态管理 | ref/reactive（无 Pinia） | 简单直接、避免过度设计 |
