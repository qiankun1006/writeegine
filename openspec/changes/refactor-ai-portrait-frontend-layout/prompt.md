# 用户原始需求记录

## 原始输入

请基于修改这个 AI 绘画工具的前端页面，要求如下：

### 核心布局（左右分栏）
1. 整体容器：100vh 高度，Flex 布局，左右分栏，间距 20px，内边距 20px
2. 左侧控制区：固定宽度 380px，垂直滚动，包含 3 个模块：
- 提示词模块：包含"正面提示词"和"负面提示词"两个多行输入框，带占位符提示
- 参考图片模块：支持图片上传（单张），上传后显示缩略图，可删除
- 参数配置模块：默认折叠，点击展开，包含：
- 下拉选择框（模型选择：SD 1.5/SDXL/Realistic）
- 滑块（图生图强度：0-1）
- 下拉选择框（采样器：Euler/Auto）
- 输入框（迭代步数：默认 20）
3. 右侧预览区：占剩余宽度，Flex 纵向布局：
- 预览区域：占 90% 高度，边框虚线，默认显示"点击生成查看结果"提示文字，图片生成后替换为图片（居中显示）
- 操作区域：占 10% 高度，包含"生成图片"按钮（主按钮）和"历史记录"文字按钮

### 样式要求
1. 整体风格简洁，主色调：浅灰（控制区）、纯白（预览区），按钮主色 #409EFF
2. 输入框/按钮有 hover 效果，圆角 8px
3. 适配 PC 端，最小宽度 1200px

### 交互要求
1. 参考图片上传后实时显示缩略图，点击删除按钮可清空
2. 参数配置模块默认折叠，点击标题展开/收起
3. "生成图片"按钮点击后显示加载状态，模拟 3 秒后显示示例图片

### 代码要求
1. 使用 Vue 3 <script setup> 语法
2. 引入 Element Plus 组件（ElInput/ElUpload/ElButton/ElCollapse/ElSelect 等）
3. 代码带清晰注释，结构分层
4. 样式使用 scoped，避免全局污染
