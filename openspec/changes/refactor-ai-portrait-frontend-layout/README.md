## ✨ 核心设计决策

### 1️⃣ 布局方案
- **选择**：Flexbox 双栏（左固定 380px，右自适应）
- **理由**：简洁、直观、浏览器兼容好
- **代码示例**：见 design.md 第 2 章

### 2️⃣ 组件结构
- **提示词模块**：来自 CoreParamsPanel.vue，保留扩展
- **图片上传模块**：ElUpload 组件，支持拖拽和预览
- **参数配置模块**：ElCollapse 组件，默认折叠，包含 4 个参数
- **预览面板**：改造 ResultsPanel.vue，上方预览、下方按钮

### 3️⃣ 技术栈
- **Vue 3 `<script setup>` 语法**（现代化）
- **Element Plus 组件库**（统一风格）
- **SCSS 主题变量**（易于维护）
- **TypeScript 类型**（代码质量）

### 4️⃣ 样式系统
- **颜色**：浅灰 #F5F5F5（左）/ 纯白 #FFFFFF（右） / 蓝 #409EFF（按钮）
- **圆角**：统一 8px
- **间距**：20px（大）/ 16px（中）/ 12px（小）
- **动画**：0.2-0.5s 过渡，平滑自然

## 🎨 样式预览（文字描述）

### 左侧控制区（380px 宽）
- 背景：浅灰 #F5F5F5
- 内边距：16px
- 三个模块：
  1. **提示词**（始终展开）
     - 标签字体大小 14px，字重 500
     - 两个文本框，高度 80px 各，多行输入
     - 边框 1px solid #DCDFE6，圆角 8px
     - 聚焦时：边框 #409EFF，阴影 0 0 0 3px rgba(64,158,255,0.2)

  2. **参考图片**（始终展开）
     - 上传区：虚线边框，浅灰背景，高度 120px
     - 缩略图：200×200 px，展示图片预览，右上角删除按钮

  3. **高级参数**（默认折叠）
     - 标题可点击，右侧箭头
     - 展开时显示 4 个参数，间距 12px
     - 每个参数：标签 + 输入控件，排列清晰

### 右侧预览区（自适应宽度）
- 背景：纯白 #FFFFFF
- 分为两层：
  1. **预览内容**（90%）
     - 虚线边框 2px dashed #BFBFBF，圆角 8px
     - 中央显示"点击生成查看结果"（浅灰文字）
     - 图片生成后，图片居中显示，自适应宽度，淡入动画

  2. **操作区**（10%）
     - 边框线 1px solid #F0F0F0
     - "生成图片"按钮：宽度 50-70%，高度 40px，蓝色 #409EFF，圆角 8px，hover 效果
     - "历史记录"按钮：文字按钮，靠右

## 📞 Q&A

**Q: 为什么不支持移动端响应式？**
A: 本轮优先优化 PC 端体验（AI 立绘生成的主要使用场景）。移动端适配留给后续迭代，UI 组件库已支持响应式，扩展相对容易。

**Q: 为什么使用 Element Plus 而不是自定义 UI？**
A: Element Plus 是业界标准、文档完善、团队熟悉。自定义 UI 成本高、维护负担重，不必要。

**Q: 什么时候开始实现？**
A: 待提案审批通过后立即开始。如有问题可在本 issue 中讨论。

**Q: 后端 API 需要改动吗？**
A: 不需要。前端仅是布局优化，API 接口保持不变。
