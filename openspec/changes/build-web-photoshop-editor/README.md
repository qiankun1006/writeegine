### 4. **design.md** ⭐️ 【实现指南】
这是详细的设计文档，包含第一阶段的核心代码设计：

#### 系统架构（核心组件代码）
1. **EventBus** - 事件系统
   - 发布/订阅模式实现
   - 事件监听和触发

2. **StateManagement** - 状态管理
   - Zustand 集成
   - 编辑器状态模型

3. **CommandHistory** - 撤销/重做系统
   - Command 基类
   - 历史栈管理
   - undo/redo 实现

4. **Layer & Document** - 图层系统
   - Layer 类定义
   - Document 类管理
   - 图层合成

5. **ToolManager & Tool** - 工具框架
   - Tool 基类接口
   - BrushTool 实现示例
   - ToolManager 工具管理

6. **CanvasRenderer** - 渲染引擎
   - Canvas 2D 渲染
   - WebGL 高性能渲染（未来）
   - 视口管理

#### HTML UI 结构
- 完整的编辑器 HTML 布局
- 工具栏、面板、画布区域设计

#### 核心流程图
- 绘制流程
- 工具切换流程

#### 第一阶段实现细节
- 必需组件列表
- 核心类详细代码

#### 后端 API 设计
- 文档保存接口
- 文档加载接口
- 图片导出接口
- 文件上传接口

## 🛠️ 技术栈

### 前端
- **框架**：React 或 Vue 3
- **状态管理**：Zustand 或 Pinia
- **画布**：Canvas API（2D） + WebGL（3D/高性能）
- **并发**：Web Worker
- **库**：Fabric.js（可选）、Three.js（3D）

### 后端
- **框架**：Node.js + Express（或继续使用Spring Boot）
- **数据库**：MongoDB 或 PostgreSQL
- **实时通信**：WebSocket
- **文件存储**：S3/OSS
