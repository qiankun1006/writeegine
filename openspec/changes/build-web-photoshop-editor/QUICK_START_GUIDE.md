## 🎨 基础操作

### 工具使用

#### 1. 画笔工具 (🖌️)
- **快捷键**: B
- **操作**: 点击激活，在画布上拖拽绘制
- **参数**:
  - 大小 (Size): 1-100 px
  - 不透明度 (Opacity): 0-100%
  - 硬度 (Hardness): 0-100
  - 颜色 (Color): 前景色选择器

#### 2. 铅笔工具 (✏️)
- **快捷键**: P
- **操作**: 硬边缘绘制，效果类似钢笔
- **参数**: 大小, 不透明度, 颜色

#### 3. 橡皮工具 (🧹)
- **快捷键**: E
- **操作**: 擦除像素，透明度擦除
- **参数**: 大小, 不透明度, 硬度

#### 4. 矩形选择 (⬜)
- **快捷键**: R
- **操作**: 拖拽绘制矩形选区
- **显示**: 蓝色虚线框表示选区

#### 5. 椭圆选择 (⭕)
- **快捷键**: E
- **操作**: 拖拽绘制椭圆选区

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| **Ctrl/Cmd + Z** | 撤销 |
| **Ctrl/Cmd + Y** | 重做 |
| **Ctrl/Cmd + S** | 保存文档 |
| **B** | 激活画笔 |
| **P** | 激活铅笔 |
| **E** | 激活橡皮 |
| **R** | 激活矩形选择 |
| **滚轮** | 缩放 |

### 图层管理

1. **创建图层**
   - 点击左侧面板 "➕ 新建" 按钮
   - 自动创建新图层并选中

2. **选择图层**
   - 在左侧列表中点击要选择的图层
   - 选中的图层显示为蓝色

3. **删除图层**
   - 选中图层后点击 "🗑️ 删除" 按钮
   - 至少保留 1 个图层

4. **调整透明度**
   - 使用图层旁的滑块调整不透明度
   - 范围: 0% (完全透明) - 100% (完全不透明)

5. **显示/隐藏**
   - 点击图层前面的复选框
   - ✓ = 显示
   - □ = 隐藏

### 色彩选择

**前景色** (用于绘制):
- 点击工具栏左上角的颜色方块
- 选择颜色确认

**背景色** (备用):
- 点击工具栏右下角的颜色方块

### 缩放和平移

**缩放方式**:
1. 鼠标滚轮向上 = 放大
2. 鼠标滚轮向下 = 缩小
3. 使用顶部 "−" "+" 按钮
4. 输入缩放百分比
5. 点击 "适应" 自动适配窗口

**平移方式**:
- 按住空格键 + 拖拽鼠标

### 文件操作

#### 导出为图片
1. 点击底部 "📥 导出" 按钮
2. 弹出菜单，选择格式:
   - **PNG** - 无损格式，支持透明度
   - **JPG** - 有损格式，文件较小
3. 图片自动下载

#### 保存文档
1. 点击底部 "💾 保存" 按钮
2. 文档保存为 JSON 格式
3. 包含所有图层和元数据
4. 可用来恢复编辑状态

#### 加载文档
1. 在浏览器中打开保存的 JSON 文件
2. 编辑器恢复所有内容

## 🎯 常见任务

### 绘制简单图形

**绘制线条**:
1. 激活画笔工具
2. 按住鼠标，拖拽绘制
3. 松开鼠标完成

**绘制圆形**:
1. 激活椭圆选择工具
2. 拖拽绘制圆形选区
3. 使用画笔在选区内绘制

### 修改已绘制内容

**纠正错误**:
1. 使用橡皮工具擦除
2. 或使用 Ctrl+Z 撤销

**调整透明度**:
1. 创建新图层
2. 调整图层透明度滑块
3. 在新图层上绘制

### 合并图层

**方法**:
1. 选择底层
2. 使用顶部菜单的合并选项 (待实现)

或

1. 直接删除不需要的图层
2. 保留需要的图层

### 导出作品

**完整教程**:
1. 完成编辑
2. 点击 "📥 导出" 按钮
3. 选择 PNG (保留质量) 或 JPG (文件小)
4. 图片自动下载

## 🐛 故障排除

### 画布白屏
- **原因**: JavaScript 加载失败或浏览器不支持
- **解决**: 刷新页面，检查浏览器控制台

### 绘制不响应
- **原因**: 未选择工具或未选择图层
- **解决**: 确保激活了工具，且选中了图层

### 导出失败
- **原因**: 浏览器安全设置
- **解决**: 允许浏览器下载，或使用不同浏览器

### 性能缓慢
- **原因**: 图层太多或图片太大
- **解决**: 删除不需要的图层，降低分辨率
