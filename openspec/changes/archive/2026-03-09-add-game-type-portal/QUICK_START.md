# 快速启动指南

欢迎使用游戏类型门户和编辑器！本指南将帮助您快速上手。

---

## 🚀 快速开始

### 1. 启动应用

```bash
# 进入项目目录
cd /Users/qiankun96/Desktop/meituan/writeengine

# 启动应用
mvn spring-boot:run
```

等待应用启动完成，看到类似以下日志表示启动成功：

```
Started WriteEngineApplication in X seconds
```

---

### 2. 访问门户

打开浏览器，访问：

```
http://localhost:8080/create-game/unity
```

您将看到游戏类型选择门户，包含四个游戏类型卡片。

---

## 🎮 创建您的第一个游戏

### 步骤 1: 选择游戏类型

在门户页面，点击任意一个游戏类型卡片上的"新建游戏"按钮。

### 步骤 2: 填写游戏信息

在弹出的对话框中：
- **游戏名称**: 输入您的游戏名称
- **游戏描述**: 输入简短描述（可选）
- 点击"创建"按钮

### 步骤 3: 开始编辑

系统将自动为您创建游戏项目，并跳转到对应的编辑器页面。

## 📝 编辑器快速操作

### 通用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Delete` | 删除选中对象 |
| `F11` | 全屏切换 |

### 2D 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| `G` | 移动工具 |
| `R` | 旋转工具 |
| `S` | 缩放工具 |
| `Space` | 平移视图 |
| `+` / `-` | 缩放视图 |

### 3D 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| `W` | 移动工具 |
| `E` | 旋转工具 |
| `R` | 缩放工具 |
| `Alt` + 拖动 | 旋转视图 |
| `中键` + 拖动 | 平移视图 |
| `滚轮` | 缩放视图 |

## 📂 编辑器类型说明

### 2D 策略战棋编辑器
**适用**: 回合制策略、战棋、塔防
**核心工具**: 网格系统、寻路编辑、单位放置

### 2D 恶魔城编辑器
**适用**: 类银河恶魔城、横版动作、平台跳跃
**核心工具**: 平台编辑、碰撞体编辑、物理预览

### 2D RPG 编辑器
**适用**: 2D RPG、JRPG、冒险游戏
**核心工具**: NPC 编辑、对话系统、任务系统

### 3D 射击编辑器
**适用**: FPS、TPS、3D 动作
**核心工具**: 3D 场景编辑、光照系统、粒子效果

## 🔧 常见问题

### Q: 如何保存游戏？
A: 点击工具栏的"保存"按钮，或使用快捷键 `Ctrl+S`。

### Q: 如何打开已有游戏？
A: 在门户页面，点击对应游戏类型卡片上的"打开游戏"下拉菜单，选择要打开的游戏。

### Q: 编辑器支持哪些浏览器？
A: 推荐使用 Chrome 90+、Firefox 88+、Safari 14+、Edge 90+。

### Q: 性能不佳怎么办？
A: 3D 编辑器可以通过点击"性能"按钮查看性能指标。减少场景中的对象数量可以提升性能。
