# Spec: 粒子编辑器核心功能

**Capability**: `particle-editor-core`
**Status**: 🎯 Proposal
**Version**: 1.0

---

## ADDED Requirements

### Requirement: 实时粒子预览

系统 SHALL 提供实时粒子预览功能。

#### Scenario: 用户打开编辑器页面时

1. 页面加载粒子编辑器应用
2. Canvas 渲染区默认加载 "火焰" 预设
3. 粒子效果在 Canvas 上实时播放
4. 显示当前粒子数量和 FPS（右下角统计信息）

#### Scenario: 用户修改参数时

1. 用户在参数面板调整任意参数（如发射速率、粒子寿命等）
2. Canvas 中的粒子效果实时更新（无需手动刷新）
3. 参数变化在 300ms 内反映到视觉效果
4. FPS 保持在 60（或接近 60）

#### 接受标准
- Canvas 渲染 1000+ 粒子时帧率 ≥ 55 FPS
- 参数更新延迟 ≤ 300ms
- 无明显卡顿或闪烁

---

### Requirement: 参数编辑面板

系统 SHALL 提供参数编辑面板功能。

#### Scenario: 参数分组折叠

1. 参数面板包含以下分组：
   - 基础参数（发射位置、发射率、角度）
   - 物理参数（初速度、加速度、重力）
   - 外观参数（颜色、透明度、缩放、旋转）
   - 纹理参数（纹理选择、TextureRegion）

2. 每个分组可独立折叠/展开
3. 折叠状态保存到 localStorage，页面刷新后保持

#### Scenario: 参数输入控件

1. 数值参数：滑块 + 文本输入框（便于精确输入）
2. 颜色参数：颜色选择器（ColorPicker）
3. 角度参数：范围滑块（0-360°）
4. 枚举参数：下拉选择框

#### Scenario: 参数范围验证

1. 用户输入超出范围的值时，显示错误提示
2. 系统自动将其限制在有效范围内
3. 错误信息清晰明了（如 "发射速率需在 1-1000 范围内"）

#### 接受标准
- 所有参数面板功能按规范实现
- 参数变化立即反映到 Canvas（无延迟）
- 参数验证正确，提示信息清晰
- UI 美观度满足深色主题要求

---

### Requirement: 预设系统

系统 SHALL 提供预设系统功能。

#### Scenario: 加载内置预设

1. 编辑器提供以下内置预设：
   - 火焰（Flame）：红黄渐变，向上，快速衰减
   - 雪花（Snow）：白色，缓慢下落，有旋转
   - 雨水（Rain）：蓝色，快速下落，直线运动
   - 爆炸（Explosion）：橙红色，快速散开，快速衰减

2. 用户点击预设名称或缩略图，一键加载预设
3. 加载后，Canvas 立即播放预设效果
4. 参数面板更新为预设参数

#### Scenario: 预设缩略图预览

1. 预设列表显示小缩略图（预设效果的静止截图）
2. 鼠标悬停时显示预设名称和描述
3. 支持预设分类（效果类型、应用场景）

#### 接受标准
- 内置 4 个预设可正常加载
- 预设视觉效果与预期相符
- 加载预设后，参数面板数据正确更新

---

### Requirement: 基础 Canvas 渲染

系统 SHALL 提供基础 Canvas 渲染功能。

#### Scenario: 高 DPI 屏幕适配

1. 在高 DPI 屏幕（Retina 屏）上，粒子渲染清晰锐利
2. Canvas 逻辑尺寸 = 物理尺寸 / devicePixelRatio
3. 无模糊、像素化问题

#### Scenario: 粒子渲染模式

1. 支持两种混合模式：
   - Normal（正常）：粒子透明度叠加
   - Additive（加法）：模拟光效，多个粒子相加变亮

2. 用户可在设置中切换渲染模式
3. 效果实时更新

#### Scenario: 背景颜色自定义

1. 用户可设置 Canvas 背景颜色
2. 支持透明背景（用于导出）
3. 背景颜色变化立即反映

#### 接受标准
- Canvas 渲染清晰，无锯齿
- 粒子混合模式正确
- 背景颜色切换无闪烁

---

### Requirement: 动画播放控制

系统 SHALL 提供动画播放控制功能。

#### Scenario: 播放/暂停/重置

1. 编辑器顶部提供控制按钮：
   - ▶ 播放（如果已暂停）
   - ⏸ 暂停
   - ⊙ 重置（清除所有粒子，从头开始）

2. 点击 "暂停" 时，粒子停止运动，但保持显示
3. 点击 "重置" 时，所有粒子消失，计数器清零
4. 点击 "播放" 时，继续发射新粒子

#### Scenario: 循环播放

1. 默认连续循环播放（发射器持续发射）
2. 支持单次播放模式（发射器发射 1 次后停止）

#### 接受标准
- 播放控制按钮功能正确
- 暂停状态保留粒子显示
- 重置清除所有粒子

## Implementation Notes

### 技术选型
- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **UI 组件库**：Element Plus
- **Canvas API**：原生 Canvas 2D
- **状态管理**：Pinia（可选）

### 性能目标
- 1000 粒子 @ 60 FPS
- 参数更新延迟 ≤ 300ms
- 内存占用 ≤ 100 MB

### 浏览器兼容性
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
