## ADDED Requirements
### Requirement: 游戏UI组件库 - 进度条组件
系统SHALL提供基于Vue3 + Composition API的进度条组件，支持6种游戏场景常用效果，具备完整的配置选项和动画效果。

#### Scenario: 访问进度条演示页面
- **WHEN** 用户访问 `/create-game/asset/progress-bar-demo`
- **THEN** 页面显示6种进度条效果的演示和对比
- **AND** 每种效果都有实时预览和参数调整面板

#### Scenario: 使用极简纯色款进度条
- **WHEN** 用户选择"极简纯色款"类型
- **THEN** 进度条显示纯色背景和填充
- **AND** 支持自定义背景色、填充色、圆角、边框样式
- **AND** 进度变化时有平滑过渡动画

#### Scenario: 使用游戏血条款进度条
- **WHEN** 用户选择"游戏血条款"类型
- **THEN** 进度条显示线性渐变（深红→鲜红）
- **AND** 包含血滴纹理效果
- **AND** 血量变化时有受损闪烁动画
- **AND** 血量低于20%时显示警示抖动效果

#### Scenario: 使用能量条款进度条
- **WHEN** 用户选择"能量条款"类型
- **THEN** 进度条显示径向渐变（蓝色→紫色）
- **AND** 有流光动画效果
- **AND** 充能完成时（进度100%）显示脉冲效果

#### Scenario: 使用加载进度款进度条
- **WHEN** 用户选择"加载进度款"类型
- **THEN** 进度条显示条纹动效
- **AND** 实时显示进度百分比数字
- **AND** 加载完成时显示✓图标，失败时显示✗图标

#### Scenario: 使用复古像素款进度条
- **WHEN** 用户选择"复古像素款"类型
- **THEN** 进度条显示8bit像素风格边框
- **AND** 填充以8px为阶梯增长
- **AND** 使用像素字体显示进度数值

#### Scenario: 使用玻璃拟态款进度条
- **WHEN** 用户选择"玻璃拟态款"类型
- **THEN** 进度条显示毛玻璃背景效果
- **AND** 填充为半透明效果
- **AND** 边框有模糊效果，适配现代游戏UI风格

#### Scenario: 编辑血条参数
- **WHEN** 用户在血条编辑面板
- **THEN** 支持拖拽滑块调整血量值（0-100）
- **AND** 支持输入框手动输入数值
- **AND** 可配置渐变起始色/结束色
- **AND** 可切换是否显示血量数值（HP）
- **AND** 可启用/禁用低血量警示
- **AND** 可调整边框像素大小（像素款）

#### Scenario: 进度条交互反馈
- **WHEN** 用户鼠标悬停在进度条上
- **THEN** 显示tooltip，包含当前进度、关联资产名称、接口状态
- **AND** 不同状态（加载中/完成/失败）有不同视觉反馈
- **AND** 失败状态显示红色+闪烁效果
- **AND** 完成状态显示绿色+对勾图标

#### Scenario: 响应式适配
- **WHEN** 在不同设备（PC端、平板端）查看进度条
- **THEN** 进度条宽度/高度支持百分比和固定像素配置
- **AND** 在游戏编辑器界面中正常显示
- **AND** 动画速度可配置（0.2s-1s范围）

#### Scenario: 集成到素材创作页面
- **WHEN** 用户访问 `/create-game/asset` 页面
- **THEN** 页面包含"进度条组件"功能块
- **AND** 点击后跳转到进度条演示和编辑页面
- **AND** 进度条样式可保存到 `/create-game/asset` 接口
