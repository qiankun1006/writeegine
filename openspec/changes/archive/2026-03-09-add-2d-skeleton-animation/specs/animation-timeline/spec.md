# Spec: 动画时间轴编辑器

## ADDED Requirements

### Requirement: 时间轴组件
系统 SHALL 提供可交互的时间轴编辑界面。

#### Scenario: 用户在时间轴上拖拽时间指针到 1.5 秒
- **WHEN** 用户拖拽时间指针
- **THEN** 时间轴必须显示从 0 到动画总时长的时间刻度
- **AND** 时间轴必须显示当前时间指针（可拖拽）
- **AND** 时间轴必须支持缩放（放大/缩小）以显示更多或更少的细节
- **AND** 时间轴必须显示帧标记（可选，用于 24/30fps 视图）

#### Scenario: 用户在时间轴上拖拽时间指针到 1.5 秒
```
初始状态：
时间轴: [0s] ────▶ [2.5s]
              时间指针在 0.5s

用户拖拽操作后：
时间轴: [0s] ────────▶ [2.5s]
                  时间指针在 1.5s
画布预览更新到 1.5s 的帧
```

### Requirement: 关键帧轨道视图
系统 SHALL 显示每个骨骼的关键帧轨道。

#### Scenario: 用户在关键帧轨道上操作
- **WHEN** 用户查看关键帧轨道
- **THEN** 时间轴下方必须显示多条轨道（一条对应一个骨骼）
- **AND** 每条轨道必须显示该骨骼的关键帧（菱形或方块标记）
- **AND** 关键帧标记必须按时间位置精确显示
- **AND** 关键帧必须支持选择、拖拽和删除

#### Scenario: 用户在关键帧轨道上操作
```
轨道视图：
骨骼: Root    ◇ ────────────────── ◇
骨骼: Arm     ◇ ─────◇───────◇─── ◇
骨骼: Hand    ◇ ──────────────◇── ◇
```

### Requirement: 关键帧设置
系统 SHALL 允许用户设置和删除关键帧。

#### Scenario: 用户在第 30 帧为 Arm 骨骼创建关键帧
- **WHEN** 用户创建关键帧
- **THEN** 用户必须能够在当前时间指针位置创建关键帧
- **AND** 用户必须能够选择已有的关键帧
- **AND** 用户必须能够删除关键帧
- **AND** 删除关键帧时可以通过撤销恢复

#### Scenario: 用户在第 30 帧为 Arm 骨骼创建关键帧
```
1. 选择 Arm 骨骼
2. 将时间指针拖到 30 帧位置
3. 在画布上调整骨骼位置/旋转
4. 点击"插入关键帧"按钮
5. 关键帧出现在轨道上
```

### Requirement: 播放控制
系统 SHALL 提供动画播放的完整控制。

#### Scenario: 用户以 1.5x 速度循环播放动画
- **WHEN** 用户控制动画播放
- **THEN** 用户必须能够点击"播放"按钮开始播放
- **AND** 用户必须能够点击"暂停"暂停播放
- **AND** 用户必须能够点击"停止"停止并重置到起点
- **AND** 用户必须能够调整播放速度（0.5x - 2.0x）
- **AND** 用户必须能够切换循环播放

#### Scenario: 用户以 1.5x 速度循环播放动画
```
控制面板：
[◀◀ 上一帧] [▶ 播放] [⏸ 暂停] [■ 停止]
速度: [1.5x] 循环: [✓ 启用]
```

### Requirement: 时间显示
系统 SHALL 清晰显示当前时间和帧数。

#### Scenario: 显示动画信息
- **WHEN** 用户查看时间信息
- **THEN** 必须显示当前时间（秒）
- **AND** 必须显示当前帧数
- **AND** 必须显示总帧数
- **AND** 必须显示帧速率（FPS）

#### Scenario: 显示动画信息
```
时间: 1.25s | 第 30/60 帧 @ 24fps
```

### Requirement: 动画参数
系统 SHALL 支持调整动画的全局参数。

#### Scenario: 用户创建 30fps 的 120 帧动画
- **WHEN** 用户调整动画参数
- **THEN** 必须能够修改帧速率（FPS）
- **AND** 必须能够修改总帧数
- **AND** 必须能够启用/禁用循环播放
- **AND** 必须能够设置播放起点和终点（用于部分播放）

#### Scenario: 用户创建 30fps 的 120 帧动画
```
动画参数：
名称: [idle]
帧速率 (FPS): [30]
总帧数: [120]
总时长: [4.0s]
循环播放: [✓]
```
