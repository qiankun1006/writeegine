## MODIFIED Requirements

### Requirement: 3D 编辑器 - 现有功能保留并优化
原有的 3D Unity 编辑器 SHALL 被优化为针对 3D 射击游戏优化的版本，保留核心功能但移除不必要的 2D 特定工具。

#### Scenario: 编辑器加载性能改进
- **WHEN** 用户访问 `/create-game/unity/3d-shooter`
- **THEN** 编辑器应在 ~300-500ms 内显示初始 UI（相比原始编辑器的 ~1s 有 50% 改进）

#### Scenario: 保留 3D 核心功能
- **WHEN** 3D 编辑器加载完成
- **THEN** 系统保留以下功能：
  - 3D 场景编辑（Three.js 渲染）
  - 摄像机控制（OrbitControls）
  - 物体创建和编辑
  - Gizmo 变换工具（移动、旋转、缩放）
  - 基础属性检查器（位置、旋转、缩放、材质）

#### Scenario: 移除 2D 特定工具
- **WHEN** 在 3D 射击编辑器中查看工具栏
- **THEN** 系统不加载以下工具（保存内存和加载时间）：
  - 网格系统（Grid System）
  - 寻路工具（Pathfinding）
  - 战斗预览（Battle Preview）
  - NPC 编辑器
  - 对话系统
  - 任务系统

### Requirement: 3D 射击游戏专用功能
系统 SHALL 为 3D 射击游戏提供特化的编辑工具和功能。

#### Scenario: 光照系统编辑
- **WHEN** 用户在 3D 射击编辑器中使用 "光照编辑" 工具
- **THEN** 系统提供光照配置面板：
  - 灯光类型：方向光、点光、聚光灯
  - 灯光强度、颜色、范围调整
  - 阴影配置
  - 环境光设置

#### Scenario: 粒子系统编辑
- **WHEN** 用户在 3D 射击编辑器中为对象添加粒子效果
- **THEN** 系统显示粒子配置面板：
  - 发射形状
  - 发射速度
  - 粒子生命周期
  - 颜色渐变

#### Scenario: 射线检测工具
- **WHEN** 用户在 3D 射击编辑器中配置射击检测
- **THEN** 系统提供射线检测编辑器：
  - 射线起点、方向配置
  - 检测范围可视化
  - 碰撞层配置
  - 预览射线路径

#### Scenario: 物理引擎延迟加载
- **WHEN** 用户首次在 3D 射击编辑器中使用物理功能
- **THEN** 系统异步加载 Ammo.js（懒加载），不阻塞初始渲染

#### Scenario: 网络同步配置（高级功能）
- **WHEN** 用户在 3D 射击编辑器的 "高级" 菜单中访问网络配置
- **THEN** 系统提供（可选功能，异步加载）：
  - 网络实体标记
  - 同步属性选择
  - 预测算法选择

### Requirement: 后向兼容性
系统 SHALL 保证现有的 3D 编辑器数据和场景可以在新的 3D 射击编辑器中正常加载和编辑。

#### Scenario: 加载现有 3D 场景
- **WHEN** 用户在 3D 射击编辑器中打开已有的 3D 场景
- **THEN** 场景完整加载，所有对象、材质、光照设置保持不变

#### Scenario: 数据格式兼容性
- **WHEN** 用户从原始 3D 编辑器切换到 3D 射击编辑器
- **THEN** 所有 UnityScene、UnityObject、UnityMaterial 数据无需转换即可使用
