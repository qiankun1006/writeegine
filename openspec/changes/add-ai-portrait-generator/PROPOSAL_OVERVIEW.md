## 🎨 功能特性

### ✨ 核心功能
- ✅ 文本提示词输入（1-500 字符，实时字数统计）
- ✅ 负面提示词输入（预填充常用负面词）

## 💡 设计亮点

1. **参数区分化设计**
   - 核心参数：必填，放在左侧面板，始终可见
   - 高级参数：可选，用折叠面板隐藏，降低认知负载

2. **渐进式增强**
   - 基础用户只需填写提示词和点击生成
   - 高级用户可调整 8 个高级参数实现精细控制

3. **完整的加载反馈**
   - 进度条显示绝对进度（0% → 100%）
   - 粒子动效提供视觉趣味
   - 预估时间文本给予时间预期

4. **考周全的错误处理**
   - 前端验证所有参数
   - 网络错误可重试
   - API 错误返回用户友好提示

5. **现代化的 UX**
   - 聚焦发光动效
   - 按钮 Hover 渐变
   - 结果卡片淡入动画
   - 参数区平滑展开/收起

## 🎓 关键决策理由

### 为什么选择 Vue 3？
- 更好的组件化能力
- Composition API 提供灵活的逻辑复用
- 生态成熟，社区活跃

### 为什么选择 Element Plus？
- 提供丰富的预制组件
- 主题定制灵活
- 与项目已有使用保持一致

### 为什么采用参数分化设计？
- 降低初级用户的认知负载
- 为高级用户保留精细控制
- 符合渐进式增强原则

### 为什么重视加载动画？
- AI 生成往往耗时较长
- 加载动画提供反馈，防止用户困惑
- 粒子动效提升产品质感
