# 原始需求 - AI 人物立绘生成工具前端页面

针对 http://localhost:8083/create-game/asset/character-portrait 页面，也就是【角色立绘】帮我按下面的需求开发，老的可以删了。

## 需求：生成AI人物立绘生成工具的前端页面

### 技术栈：

- 核心框架：Vue 3 + Vite + TypeScript

- UI组件库：Element Plus（高级定制主题）

- 样式方案：SCSS + CSS变量 + 响应式布局

- 动效：CSS过渡动画 + 轻量级Canvas动效（生成加载阶段）



### 设计风格：

- 整体风格：科技感+极简美学，主色调为深紫(#6C5CE7) + 浅灰(#F8F9FA) + 纯白(#FFFFFF)，辅助色为浅蓝(#00C4FF)，中性色为深灰(#2D3748)

- 布局：卡片式分区，留白充足，层级分明，支持响应式（PC优先，适配平板/手机）

- 动效：输入框聚焦动效、按钮hover渐变、生成过程加载动画（进度条+粒子动效）、图片预览淡入动画

- 视觉层次：核心操作区突出，高级参数区可折叠，结果展示区沉浸式



### 功能模块与交互逻辑：

1. 顶部导航区：

   - 左侧品牌名称「AI立绘生成器」，右侧用户中心（极简占位）

   - 导航栏固定，滚动时轻微透明度变化



2. 核心操作区（必选参数）：

   - 提示词输入：多行文本框，占位符提示「例如：日系二次元少女，长发粉发，猫耳，穿洛丽塔裙，纯色背景，光影柔和」，支持字数统计，聚焦时边框发光动效

   - 负面提示词输入：独立多行文本框，占位符提示「例如：模糊，多手指，水印，变形，低分辨率」，默认填充常用负面词

   - 参考图片上传：

     - 支持拖拽/点击上传，上传后显示缩略图（最大200x200），可删除重新上传

     - 提示支持格式：PNG/JPG/WEBP，大小≤10MB

   - 微调模型权重：滑块组件（0-1，步长0.01），右侧显示当前值，默认0.8

   - 尺寸选择：

     - 预设选项：512×768（竖版立绘）、1024×1024（方形）、2048×2048（高清）

     - 自定义尺寸输入框（限制为2的幂次，最小256，最大2048）

   - 生成按钮：大尺寸渐变按钮（深紫→浅蓝），点击后禁用并显示加载动画，按钮文字变为「生成中...」



3. 高级参数区（可折叠）：

   - 折叠面板默认收起，标题「高级参数」带展开/收起动效

   - 包含以下参数：

     - 图生图强度：滑块（0-1，步长0.01），默认0.6，附带说明「值越高越贴近参考图」

     - 生成数量：数字输入框（1-5，默认1）

     - 采样器选择：下拉框，选项「Euler（快速）、DPM++ 2M Karras（效果优）、AutoCFG（智能）」，默认Euler

     - 迭代步数：滑块（10-50，步长1），默认30

     - 风格预设：下拉框，选项「日系二次元、国风写实、赛博朋克、Q版卡通、无预设」，默认无预设

     - 种子值：数字输入框（-1为随机，默认-1），附带说明「固定种子可复现结果」

     - 面部修复：开关组件，默认开启

     - 输出格式：单选框，选项「PNG（透明背景）、JPG（高清)」
